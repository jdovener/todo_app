About the app

This is a simple To-do app which has full CRUD functionality


C - Create - User is able to add new to-do tasks to the list

R - Read - User is able to display all existing to-do tasks

U - Update - User is able to press a button to mark a to-do task as 'complete' or 'incomplete'

D - Delete - User is able to delete to-do tasks from the list
